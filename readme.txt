WP Image Copyright plugin for managing image copyrights.

== Description ==

This plugin allows you to manage and display image copyrights via shortcodes. You can add copyright information to images and control their display from the backend.

== Installation ==

1. Upload `wp-image-copyright.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the provided shortcodes to display image copyright information.
